CSCE 242 - Caleb Steckler

CSCE 242 is a Web Development course where we will learn the basics of HTML, CSS, and JavaScript. We will create many web pages for assignments and examples during the semester. Throughout the semester, we will be working on a project where we can create anything we want using the skills we learn

GitHub Repo

Assignments

  • Assignment 1 - Basic HTML

    • Description: In this assignment, we learned the basics of HTML structure and elements. We implemented images, tables, and text to create a simple web page.

  • Assignment 2 - Basic CSS

    Description: For the Basic CSS assignment, we learned how to style our HTML pages using CSS. We created an external stylesheet and applied various styles to elements on the page such as colors, fonts, and layouts.

  • Assignment 3 - Flexbox

    Description: In this assignment, we explored the Flexbox layout model in CSS. We created flexible and responsive layouts using flex containers and flex items to arrange content dynamically on the page.
